# ADO-1163 — NuGet Package Metadata + MIT LICENSE

## Goal

Add missing NuGet package metadata fields (`Description`, `Authors`, `PackageTags`, `PackageProjectUrl`) to `XrmGhost.Attributes.csproj` and create a `LICENSE` (MIT) file at the repo root so GitHub detects the repository license and the published package is complete for nuget.org consumers.

## Scope & Modifications

### Files Modified

**`XrmGhost.Attributes.csproj`**  
Added four new fields inside the existing `<PropertyGroup>`:
- `<Description>` — public-facing package description for nuget.org
- `<Authors>` — set to `XrmGhost`
- `<PackageTags>` — `dataverse;xrm;dynamics365;powerplatform;plugin;crm;attributes;xrmghost`
- `<PackageProjectUrl>` — `https://github.com/xrmghost/xrmghost-attributes`

Added one new entry in the existing `<ItemGroup>`:
- `<None Include="LICENSE" Pack="true" PackagePath="" />` — bundles the LICENSE file into the NuGet package root

**No version fields, framework target, signing settings, or `PackageLicenseExpression` were changed.**  
**`RepositoryUrl` was already correctly set to `https://github.com/xrmghost/xrmghost-attributes` — left untouched.**

### Files Created

**`LICENSE`** (new file at repo root)  
Standard MIT License text, year 2024, holder XrmGhost.
