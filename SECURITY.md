# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** open a public GitHub issue.

Instead, use GitHub's private vulnerability reporting feature:

1. Go to the [Security tab](../../security) of this repository.
2. Click **"Report a vulnerability"** (under *Advisories*).
3. Fill in the details and submit the report.

This allows us to review and address the issue before any public disclosure.

Critical vulnerabilities will be acknowledged within **7 days** and addressed within **30 days** of the initial report.
